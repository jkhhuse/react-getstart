## vscode with react  

#### jsx grammar support  
安装 sublime babel插件  

#### intelligence  
* step 1: install typings   
```shell
> npm install typings --global  
```
* step 2: install react ts definitions  
```shell
> typings init  
> typings install dt~react   
```
